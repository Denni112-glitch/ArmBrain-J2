# ArmBrain J2 - NEMA23 Robotic Shoulder Joint

## Specs
- **Motor:** NEMA23 stepper
- **Driver:** TB6600 stepper driver 
- **Bearing:** thin-section bearing for zero backlash
- **Controller:** Arduino Uno R3
- **Material:** PETG-CF + Aluminum
